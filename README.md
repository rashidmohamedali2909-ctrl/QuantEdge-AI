## Overview

QuantEdge AI is a project where I explored how AI can be used to analyze financial data and support basic trading decisions.

The system is built using multiple agents, where each handles a specific task like sentiment analysis, valuation, or risk evaluation. These outputs are then combined to simulate a simple decision-making process.

⚠️ This project is for learning purposes only and does not perform real trading.
